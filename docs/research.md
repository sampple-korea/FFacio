# Runtime 판정 정책 메모

FFacio 0.6.1의 얼굴 입력과 판정은 제공된 FFacio Runtime Demo 구현을 기준으로 정리했습니다. 앱은 Runtime 템플릿을 해석하거나 자체 임베딩으로 대체하지 않습니다.

기본값은 식별 기준 0.80, 1위·2위 차이 0.03, 라이브니스 0.70, 품질 0.50, 양쪽 눈 0.80, 가림·입 0.50, yaw/pitch/roll 10도, 등록 안정화 1200ms, 분석 간격 180ms입니다. 사용자 요구에 따라 인증 안정화만 Demo 기본 3프레임 대신 1프레임으로 고정했습니다.

등록은 대표 Runtime 템플릿 하나만 저장하고, 인증도 사용자당 대표 템플릿 하나만 비교합니다. 여러 얼굴이 보이면 가장 큰 얼굴만 처리합니다. 능동 동작 챌린지, 다각도 등록, 샘플 간 일관성 검사와 보조 샘플 지지 판정은 사용하지 않습니다.

이 기준은 실제 배치 환경의 카메라, 거리, 조명, Runtime 모델 버전과 사용자 집단에 따라 오인식·거부율을 따로 측정해야 합니다. 관리자 판정 로그는 최고 점수·2위 점수·거부 이유를 기록하지만 공개 운용 화면에는 생체 점수나 과거 사용자 이름을 노출하지 않습니다.
